DOCUMENT OBJECT MODEL (DOM)

selecting elements inside the dom using js.

we can select elements by various method

1.By Tag Name

This property gives for all elements with the particular tag name given .

document.getElementsByTagName("li")

it will give all element with tag li.

2.By Class Name

This property gives for all elements with the particular class name given .

document.getElementsByClassName("btn")

it will give all element of class btn.

2.By Id

This property gives for all elements with the particular id name given. Id is unique on every page. Hence we use element keyword instead of elements

document.getElementById("emp_id")

it will give all element of id emp_id.
